Ausführliche Beschreibung

Encoder-Motor als Servo steuern.

Mit einem Objekt dieser Klasse lässt sich ein Encoder-Motor wie ein Servo steuern.

Der Winkel wird in Grad (Vollkreis = 360°) angegeben.
